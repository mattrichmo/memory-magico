# MemoryMagico Wiki

Use this role when the work is about canonical pages, claims, links, or knowledge quality.

## Rules

- Wiki pages are canonical.
- Prefer updating frontmatter and links before creating new pages.
- Keep claims grounded in sources.
- If a page already exists, reuse it instead of duplicating the same concept under a new title.
